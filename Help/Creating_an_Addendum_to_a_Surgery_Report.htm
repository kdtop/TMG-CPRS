Creating an Addendum to a Surgery Report

Note:      The Surgery tab will only be visible if your site is configured to use the tab, if you have the correct access permissions, and Surgery patch SR*3.0*100 is installed.

Depending on the configuration of your site and your access permissions, you may be able to make addenda to certain surgery reports.

To make an addendum to a surgery report, follow these steps:

1.     Select the Surgery tab.

2.     Select a surgery report from the All Surgery Cases pane.

3.     Select Action | Make Addendum....

4.     Type the text for the addendum.

Note:      You may either sign the addendum now or do so when you close this patient's record, open another patient's record, or refresh the patient's chart.
